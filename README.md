# 📊 YouTube Trending Analysis

Este projeto apresenta um dashboard desenvolvido em **Tableau** para analisar tendências de vídeos no YouTube.

O objetivo é explorar métricas como visualizações, likes, categorias e identificar padrões em vídeos que se tornam trending.
